# Citilite-Traning-Cohort

**Citlite** is its own thing — a short, public-style session pack. It does **not** live inside the paid bootcamp folders and **nothing** in the main teaching tree points here.

**What this is:** a **taste** of how you *can* work with AI for a simple shipped page — spec → generate (e.g. z.ai) → quick QA → host (Vercel or Netlify).  

**What this is not:** the full program, week-by-week materials, assignments, or everything paying students get. Pull **a few** patterns you are comfortable sharing; keep the rest in the paid track.
